## 📖 课程简介

这是一套专为 **AI 初学者** 设计的 Prompt Engineering（提示词工程）公开课。课程的核心目标是：

✅ **重塑认知** - 从"随便聊聊"转变为"精准控制"的思维模式  
✅ **掌握框架** - 学会使用 CO-STAR、思维链等行业标准框架  
✅ **即学即用** - 涵盖职场、创意、学习等高频实战场景  
✅ **持续进化** - 建立自我优化 Prompt 的能力

**适合人群：**

- 刚接触 AI 工具的新手
- 想提升 AI 使用效率的职场人士
- 对 Prompt Engineering 感兴趣的学习者
- 希望系统学习 AI 应用的同学

## 💡 核心方法论

### CO-STAR 框架

```
C - Context (背景): 提供任务的背景信息
O - Objective (目标): 明确你希望 AI 完成什么
S - Style (风格): 指定输出的写作风格
T - Tone (语气): 设定情感色彩
A - Audience (受众): 明确内容是给谁看的
R - Response (格式): 规定输出的形式和结构
```

### 思维链 (Chain of Thought)

```
基础用法: "请一步步思考"
进阶用法: 提供 2-3 个示例，引导 AI 的思考方向
```
